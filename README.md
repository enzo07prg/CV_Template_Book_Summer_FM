Guide Overleaf — Créer son CV:

Pour commencer, il faut se rendre sur overleaf.com et créez un compte en cliquant sur Register. Une simple adresse e-mail suffit. 
Une fois connecté, cliquez sur New Project puis Blank Project, donnez un nom à votre projet (par exemple Prénom_NOM_CV) et le fichier main.tex s'ouvre automatiquement.
Supprimez tout le contenu présent dans ce fichier, puis copiez-collez intégralement le template CV fourni. Cliquez ensuite sur le bouton vert Recompile: votre CV s'affiche instantanément sur la droite de l'écran.
Maintenant c'est à vous de jouer : Complétez le nom, e-mail, les expériences professionnelles, les formations et les compétences. À chaque modification, recompilez pour voir le résultat. Overleaf gère automatiquement la mise en page, les marges et la typographie, vous n'avez pas à vous en préoccuper.
Une fois fini, cliquez sur Download PDF pour télécharger votre CV.
